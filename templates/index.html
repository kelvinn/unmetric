Unmetric is a free tool created by Kelvin Nicholson for monitoring the availability of websites. It was written using Python and Django.


You can sign-on now using a valid Google Account and start adding sites immediately. After you've added a site it will be polled via a HTTP HEAD request every five minutes. If your site returns a status code other than 200, it will be retested and an email will be sent to your account. A second email will be sent when your site's status changes.
